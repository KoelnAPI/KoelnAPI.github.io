WILLKOMMEN BEI KÖLN API

Köln API ist eine Gruppe von Aktiven im Kölner Raum die sich der Förderung der Bereitstellung und Nutzung von Open Data verschieben hat. Köln API geht es vor allem um Wissensaustausch, Netzwerken und Erfolgserlebnisse. Bei Köln API kann jeder mitmachen. Mit machen kann man, indem man an Treffen und Veranstaltungen teilnimmt, in der Mailingliste
